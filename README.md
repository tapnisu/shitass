# How to use

1. Install [deno](https://deno.land/#installation)

1. Set up your [.env](example.env) (example in [example.env](example.env))

```env
BOT_TOKEN="YOUR_TOKEN"
BOT_COLOR="#925dc4"
```

1. Start bot via `make start`

Now you have running bot!
